## 🚀 使用指南

### 快速开始
```bash
# 1. 安装依赖
npm install --legacy-peer-deps

# 2. 编译合约
npm run compile

# 3. 运行测试
npm test

# 4. 部署到 Sepolia
npm run deploy:full

# 5. 部署跨链功能
npm run deploy:crosschain

# 6. 测试交互
npm run deploy:interact
npm run interact:crosschain
```
